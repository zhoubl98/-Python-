### 基于Python的问题解析课程期末大作业（周柏林）
#### 报告简介
本报告的研究对象是沪深300指数和工农中建交五家银行的个股在2015—2024年的市场交易数据。报告主要包含两个部分，一是股票数据探索性分析，而是股价涨跌预测模型构建。在第一部分中，我们依次讨论了股价走势、收益和风险相关指标、以及个股的市场β值和α值。在第二部分中，我们设计了16个特征指标并利用统计方法对它们进行检验，分别建立线性回归模型和决策树模型并评估预测优良性，构建投资策略并评价市场模拟结果。
结果表明，线性回归模型的预测准确率为53.44%、查准率为58.68%，决策树模型略高于前者，分别为54.20%和58.87%。策略收益率和夏普比率也略有优势。
#### 代码运行说明
期末大作业代码是一个.ipynb文件，使用的解释器是Python3.13.7版本。使用到的包及其版本分别如下：pandas2.3.2、numpy2.3.2、matplotlib3.10.6、statsmodels0.14.5、sklearn1.8.0。打开代码文件后顺序运行代码单元格即可复现结果。
